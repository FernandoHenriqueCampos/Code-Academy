# 📚 Code Academy — Projetos Front-End

Este repositório reúne todos os projetos desenvolvidos durante o curso **Code Academy (Formação Front-End)**.

O objetivo deste espaço é documentar minha evolução prática ao longo do curso, armazenando exercícios, desafios e projetos aplicados que envolvem as principais tecnologias do desenvolvimento web.

---

## 🚀 Sobre o Curso

A formação em Front-End da Code Academy aborda fundamentos e práticas modernas do desenvolvimento web, incluindo:

- **HTML5**
- **CSS3**
- **Bootstrap**
- **JavaScript e TypeScript**
- **Git e GitHub**
- **Docker**
- **Vue.js**

---

## 🗂 Estrutura do Repositório

Cada pasta representa um projeto ou desafio específico realizado durante o curso.

### 📁 Projetos

#### 🔹 Proj01-Docker

> Criação de uma imagem e execução de um container utilizando Docker.

#### 🔹 Proj02-Site

> Desenvolvimento de um site estático com o objetivo de reforçar conhecimentos em HTML e CSS, reproduzindo um design moderno.

---

### 📦 Estrutura Interna de Cada Projeto

Dentro de cada pasta você encontrará:

- Código-fonte do projeto
- Arquivos **HTML**, **CSS** e **JavaScript**
- Recursos utilizados (imagens, fontes, etc.)
- `README.md` individual (quando necessário)
